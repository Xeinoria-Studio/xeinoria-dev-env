# Plugins à télécharger manuellement

Certains add-ons Skript ne sont pas disponibles via un CDN public (SpigotMC requiert un compte).
Téléchargez-les et déposez les fichiers `.jar` **dans ce dossier**.

Ils seront automatiquement copiés dans le serveur au démarrage du container.

| Plugin | Version prod | Source |
|--------|-------------|--------|
| skRayFall | 1.9.30 | [SpigotMC](https://www.spigotmc.org/resources/skrayfall.30946/) |
| PermSk | 2.0.0 | [SpigotMC](https://www.spigotmc.org/resources/permsk.90413/) |
| skript-placeholders | 1.7.1 | [SpigotMC](https://www.spigotmc.org/resources/skript-placeholders.57022/) |
| skUtilities | latest | [SpigotMC](https://www.spigotmc.org/resources/skutilities.7811/) |
| WorldGuard | 7.0.16 | [EngineHub](https://dev.enginehub.org/job/worldguard/lastStableBuild/) |

> **Note :** Les scripts qui utilisent ces add-ons ne fonctionneront pas sans eux,
> mais la plupart des scripts de base fonctionnent sans installation supplémentaire.
